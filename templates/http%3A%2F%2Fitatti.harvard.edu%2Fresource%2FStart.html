About the Semantic Cultural Warehouse

The Semantic Cultural Warehouse contains data from several data sources, which has been integrated and presented as a single collection to the users. More specifically, it has been derived from:

  • Florentine Drawings
  • Homeless Paintings of the Italian Renaissance
  • FotoIndex

You can search for infromation using the Structured Search facilities.
